### 파리크라상 브랜드 통합 홈페이지 Web/APP 구축

###. 1 프로젝트 개요
  1.1 프로젝트명 : 파리크라상 브랜드 통합 홈페이지 WEB/APP 구축
  1.2 핵심 컨셉 : 파리크라상의 9개 사이트의 분산된 경험을 하나의 플랫폼으로 통합
                 9개 사이트의 브랜드/정체성은 유지하면서 UXUI 일관성 확보
				 파리크라상 법인 홈페이지 분리 구축 브랜드 통합 홈페이지 Footer 연계
  1.3 목표 : 9개 사이트의 브랜드별/핵심 사업별 아이덴티티가 파편화되지 않은 하나의 유기적인 인터페이스로 통합
            단일 로그인 기반 통합 서비스 구축 및 간편 SNS 가입 제공(카카오톡/네이버/구글/애플)
			seamless 이동 UX 제공
  
### 2. 기술 스택
  2.1 Framework: Next.js 14+ (App Router)
  2.2 Styling : Tailwind CSS
  2.3 Component: Lucide React, Shadcn UI
  2.4 Layount : 반응형 Web PC, Mobile, Tablet 해상도 대응
  
### 3. UX/UI 시스템 가이드
  3.2 메인 랜딩 페이지 (paris.co.kr)
      3.2.1 9개 사이트로 이동할 수 있는 메뉴 구성과 브레드크럼 제공
	  3.2.2 9개 사이트의 주요 프로모션/이벤트를 통합 제공하는 통합 배너 디자인 구성
	  3.2.3 9개 사이트 영역별 핵심 상품 정보 제공 영역 스크롤 최소화 제공
	  3.2.4 개인화 추연 영역 차별화 구성
  3.3 9개 사이트별 상세 페이지
      3.3.1 paris.co.kr/하위 도메인 url 기반의 9개 사이트별 서비스 특화 전용 페이지 구성
	  3.3.2 하위 도메인 url에서 브랜드 통합 홈페이지로 이동 가능한 사용자 이탈 최소화 UX 구성
  3.4 파리크라상 법인 홈페이지 분리 구축
      3.4.1 Footer 영역 회사 소개 링크를 통한 법인 홈페이지 별도 페이지 이동 구성
	  3.4.2 파리크라상 법인 전용 홈페이지 메뉴 및 기능 구성 (메뉴구성 소스: https://www.sangmidang.co.kr/)
